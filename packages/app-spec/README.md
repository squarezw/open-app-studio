# @oas/app-spec

The App Spec DSL — the buildable app definition (screens, components, navigation, state, data) — and the **Blueprint Compiler** that turns an IFG (or subgraph) into a draft spec.

Also owns the patch API shared by the canvas and the AI sidebar: `proposePatch / applyPatch / revertPatch`.
